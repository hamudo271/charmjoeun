# Charmjoeun Dental Clinic - React Rebuild

This project is a React reconstruction of the original dental clinic website.
It uses Vite, React, Tailwind CSS, and React Router.

## Getting Started

1.  Open your terminal and navigate to this directory:
    ```bash
    cd dental-clinic
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn
    ```

3.  Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  Open your browser at the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

-   `src/components`: Reusable UI components (Navigation, Footer).
-   `src/pages`: Page components corresponding to each route.
-   `src/App.jsx`: Main routing configuration.
-   `tailwind.config.js`: Tailwind CSS configuration.
